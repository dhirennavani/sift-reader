# Assets Folder

This folder contains app icons and splash screens for the Expo app.

To add proper icons and splash screens:
1. Place icon.png (1024x1024)
2. Place adaptive-icon.png (1024x1024) for Android
3. Place splash.png (1284x2778) for splash screen
4. Place favicon.png (48x48) for web

You can generate these using:
- https://www.appicon.co/
- https://icon.kitchen/
